Extended Note Block

Extended Note Block 是一个专为 Minecraft Fabric 平台设计的模组，它为游戏引入了全新的音乐创作体验。该模组添加了一个功能强大的扩展音符盒 (Extended Note Block) 方块，将原版简单的音符系统升级为完整的音乐制作工具。

✨ 核心功能

  • 独立音符盒方块: 提供一个全新的“扩展音符盒”方块，配备专用 GUI，与原版音符盒系统完全独立。
  • 完整的 General MIDI 乐器库: 支持全套 128 种标准 MIDI 乐器，涵盖从钢琴、弦乐到合成音色的全部范围，通过直观菜单即可选择。
  • 可视化钢琴键盘界面: 提供涵盖 128 个琴键（C-1 到 G9）的可滚动钢琴界面，实现精确音高选择，告别重复点击。
  • 专业级音符控制参数:
    • 力度 (Velocity): 精确调节音符演奏强度，直接影响音量。
    • 延音 (Sustain): 以游戏刻为单位设置音符持续时间。
  • 基于 SoundFont 的自定义音频引擎:
    • 模组采用高效的 SoundFont (.sf2) 技术动态生成乐器声音，不含预置音频。
    • 默认集成高品质且可免费分发的 GeneralUser GS 音源库。
    • 支持用户导入自定义 .sf2 文件，实现完全个性化的音色定制。

⚙️ 工作原理

本模组通过独特的一次性生成机制创建音频资源，而非传统方式。

  1. SoundFont 技术: 使用包含乐器采样数据的 .sf2 文件作为音源基础。
  2. 外部工具集成: 模组将引导您配置两个专业音频工具：
    • FluidSynth: 高性能数字音频合成器，负责音频合成。
    • FFmpeg: 多媒体框架，负责将音频转换为 Minecraft 兼容的 .ogg 格式。
  3. 自动资源包生成: 配置完成后，模组将自动调用上述工具，为每个音符和乐器组合生成音频文件，并打包成资源包供游戏加载。

💾 安装要求与步骤

前置要求: Fabric Loader 和 Fabric API。

  1. 确保已正确安装 Fabric Loader。
  2. 下载最新版 Fabric API 并将其放入 mods 目录。
  3. 下载最新版 Extended Note Block 模组并放入 mods 目录。
  4. 启动游戏以完成安装。
