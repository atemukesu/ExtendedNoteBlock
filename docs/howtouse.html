使用教程

前置要求

在启动游戏之前，请先确保您的电脑中安装了 FFmpeg 和 FluidSynth 并正确配置了环境变量。这些工具是生成音频资源所必需的，没有它们，扩展音符盒将无法正常工作。

安装与配置 FFmpeg

Windows 系统

  1. 访问 FFmpeg 官方网站，在"Get packages & executable files"部分找到适合 Windows 的版本（推荐从gyan.dev或BtbN的链接下载）
  2. 下载后得到一个压缩包（通常是.zip或.7z格式），右键点击压缩包，选择"解压到当前文件夹"或"解压到ffmpeg-xxx"（xxx是版本号）
  3. 解压完成后，将整个文件夹移动到一个简单的位置，比如直接放在C盘根目录下（示例路径：C:\ffmpeg）。注意：路径中不要有中文或空格，否则可能出错
  4. 记住该文件夹中bin文件夹的完整路径（示例：C:\ffmpeg\bin），稍后会用到
  5. 配置环境变量：
    • 按下键盘上的 Win + E 打开文件资源管理器
    • 在左侧导航栏找到"此电脑"，右键点击它，选择"属性"
    • 在打开的窗口中，点击右侧的"高级系统设置"（如果看不到，可能需要先点击右上角的"系统信息"）
    • 在弹出的"系统属性"窗口中，点击下方的"环境变量"按钮
    • 在新窗口的"系统变量"区域中，找到名为"Path"的变量，点击它然后选择"编辑"
    • 在"编辑环境变量"窗口中，点击"新建"按钮
    • 粘贴之前记住的bin文件夹路径（例如C:\ffmpeg\bin），然后点击"确定"
    • 连续点击所有打开窗口的"确定"按钮，保存设置
  6. 验证安装是否成功：
    • 按下 Win + R，输入cmd后按回车，打开命令提示符窗口
    • 在窗口中输入ffmpeg -version并按回车
    • 如果显示了FFmpeg的版本信息（不是"不是内部或外部命令"），则安装成功

macOS 系统

  1. 首先需要安装Homebrew（如果没有的话）：
    • 打开"终端"应用（可以在启动台的"其他"文件夹中找到）
    • 复制粘贴以下命令并按回车：/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • 按照提示完成安装，可能需要输入电脑密码
  2. 安装FFmpeg：在终端中输入brew install ffmpeg并按回车，等待安装完成
  3. 验证安装：输入ffmpeg -version，如果显示版本信息则安装成功

Linux 系统

  1. 打开终端（通常可以用Ctrl + Alt + T快捷键）
  2. 根据您的Linux发行版输入相应命令：
    • Ubuntu/Debian：sudo apt update && sudo apt install ffmpeg
    • Fedora/RHEL：sudo dnf install ffmpeg
    • Arch Linux：sudo pacman -S ffmpeg
  3. 输入密码（终端输入密码时不会显示字符，输入完成按回车即可）
  4. 验证安装：输入ffmpeg -version，如果显示版本信息则安装成功

安装与配置 FluidSynth

Windows 系统

  1. 访问 FluidSynth 发布页，
  2. 在Assets中下载适合Windows系统的最新版本（通常是zip格式的压缩包）
  3. 解压压缩包到一个简单路径，例如C:\fluidsynth（同样避免中文和空格）
  4. 找到并记住bin文件夹的路径（示例：C:\fluidsynth\bin）
  5. 配置环境变量（步骤与FFmpeg相同）：
    • 通过"此电脑"→"属性"→"高级系统设置"→"环境变量"
    • 在"系统变量"中找到"Path"并编辑
    • 点击"新建"，粘贴FluidSynth的bin文件夹路径
    • 保存所有设置
  6. 验证安装：
    • 打开命令提示符（Win + R→输入cmd→回车）
    • 输入fluidsynth --version并按回车
    • 显示版本信息则安装成功

macOS 系统

  1. 打开终端，确保已安装Homebrew（安装方法见FFmpeg部分）
  2. 输入命令：brew install fluidsynth并按回车
  3. 验证安装：输入fluidsynth --version，显示版本信息则成功

Linux 系统

  1. 打开终端
  2. 根据发行版输入命令：
    • Ubuntu/Debian：sudo apt install fluidsynth
    • Fedora/RHEL：sudo dnf install fluidsynth
    • Arch Linux：sudo pacman -S fluidsynth
  3. 验证安装：输入fluidsynth --version，显示版本信息则成功

渲染核心数设置

在配置界面中，您需要设置渲染线程数量，以下是一些建议：

  • 建议设置为不超过您电脑的 CPU 核心数
  • 对于 4 核 CPU，建议设置为 2-3 线程
  • 对于 8 核 CPU，建议设置为 4-6 线程
  • 线程数过高可能导致系统卡顿，但会加快渲染速度
  • 线程数过低会减慢渲染速度，但不会影响系统稳定性

首次渲染可能需要较长时间（数分钟到数十分钟），请耐心等待。渲染完成后，您就可以正常使用扩展音符盒了。
